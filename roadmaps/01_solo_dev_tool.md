# Path 1 - Solo Developer Tool

## Objective
Bootstrap CODE into a profitable solo developer product with low risk and fast feedback loops.

## 6-Month Targets
- 100 active users
- 10 paying users
- $500+ MRR
- Stable release pipeline (PyPI + GitHub)

## Week 1-2 (Execution Start)
1. Package for PyPI.
2. Prepare launch posts (Reddit, Hacker News, Dev.to).
3. Set up first-100-users tracker + feedback loop.

## Current Execution Status
- [x] PyPI packaging baseline added (`pyproject.toml`, package entry point).
- [x] Launch templates created (`docs/launch/path1_outreach.md`).
- [x] First-100 tracker created (`docs/launch/first_100_users.csv`).

## Month 1
1. Publish alpha release.
2. Collect first 20 user conversations.
3. Prioritize top 3 pain points and ship weekly fixes.

## Month 2-3
1. Add VS Code extension prototype.
2. Improve onboarding (`--help`, examples, docs).
3. Add basic telemetry opt-in via local report command.

## Month 4-6
1. Introduce paid tier (team policies / premium checks).
2. Improve retention with pre-commit + CI guides.
3. Convert top free users to paid plan.

## Exit Criteria
- If >= $5K MRR, consider Path 2 or 3.
- If user growth plateaus, reevaluate positioning and distribution.
