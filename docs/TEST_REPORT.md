## ✨ Feature Highlights

### 1. Parallel Processing
- **Auto-detects CPU cores** (uses N-1 cores)
- **Custom worker count** (`--workers 4`)
- **Progress tracking** (tqdm integration)
- **3-10x speedup** on large codebases

### 2. Pre-Commit Hook
- **One-command install** (`--install-hook`)
- **Auto-checks staged files** (Python only)
- **Prevents bad commits** (exit code 1 if normalization needed)
- **Easy bypass** (`git commit --no-verify`)

### 3. Incremental Processing
- **SHA256 hash tracking** (cryptographically secure)
- **Skip unchanged files** (90%+ skip rate typical)
- **Persistent cache** (`.normalize-cache.json`)
- **10-100x faster** on repeated runs

### 4. Multi-Language Syntax
- **8 languages supported** (Python, JS, TS, Go, Rust, C, C++, Java)
- **Graceful fallback** (if checker not installed)
- **Timeout protection** (10s per check)
- **Non-fatal errors** (continues processing)

### 5. Interactive Mode
- **File-by-file approval** (y/n/d/q commands)
- **Diff preview** (shows first 10 changes)
- **Full diff on demand** (d command)
- **Quit anytime** (q command)
