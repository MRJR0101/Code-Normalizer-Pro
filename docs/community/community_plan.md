# Community Plan (Phase 3)

## Objective
Create repeatable community touchpoints to support user acquisition and retention.

## Weekly Cadence
1. Publish one practical tip post with CLI examples.
2. Reply to all open feedback threads from prior week.
3. Log feedback in `docs/launch/first_100_users.csv`.
4. Regenerate pain-point summary:
   `python scripts/feedback_prioritizer.py`

## Contribution Surface
- Issues via GitHub templates.
- PRs guided by `CONTRIBUTING.md`.
- Changelog updates for visible progress.
